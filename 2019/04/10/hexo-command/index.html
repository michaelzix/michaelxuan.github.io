hexo常见的命令

发表于 2019-04-10 更新于 2019-11-03 分类于 hexo 阅读次数：

记录hexo常见的命令

init

hexo init [folder]  初始化一个新的项目

new

hexo new [layout] <title>  添加一个新文章

generate 简写 g

hexo generate 生成静态文件

server

hexo server [-p 端口]  启动服务器。默认情况下，访问网址为： http://localhost:4000/。

deploy 简写 d

hexo deploy 发布到服务器上

clean

hexo clean  清理已经生成的静态文件
